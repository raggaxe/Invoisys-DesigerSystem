O portal Invoisys usa .btn em componentes.css com modificadores .btn-primary, .btn-band, .btn-canceled e .btn-link (texto sublinhado, fundo transparente) como referência visual para UI nova. A classe .btn-secondary mantém-se no CSS por compatibilidade com telas antigas; nesta documentação não a promovemos como padrão. Não replicamos o catálogo completo do Bootstrap 5.3 (success, outline, tamanhos lg/sm, etc.); a referência serve para padrões de marcação (ícone + texto, link com aspecto de botão, desativado, menu). Incluímos ainda chips de toolbar (Mostrar Métricas / Filtrar por em table.css) e o botão só ícone com borda (btn-outline-icon).

Referência: variantes e padrões gerais em Bootstrap 5.3 · Buttons. Nesta doc, o menu pendente usa classes invoisys-ds-dropdown* e o script em design-system-docs.js só para a demonstração; em produção você pode integrar ao seu JS (ex. Bootstrap) mantendo o mesmo HTML de botões e itens.

Variantes

Preferir type="button" em formulários para não submeter por engano.

HTML 
<button type="button" class="btn btn-primary">Primário</button>
<button type="button" class="btn btn-band">Marca (band)</button>
<button type="button" class="btn btn-canceled">Cancelar</button>

Com ícones

O .btn já define display: inline-flex e gap entre ícone e rótulo. Use Font Awesome com aria-hidden="true" quando o ícone for puramente decorativo.

HTML 
<button type="button" class="btn btn-primary">
    <i class="fa fa-check" aria-hidden="true"></i> Guardar
</button>

<button type="button" class="btn btn-canceled">
    <i class="fa fa-search" aria-hidden="true"></i> Pesquisar
</button>

<button type="button" class="btn btn-band" aria-label="Adicionar linha">
    <i class="fa fa-plus" aria-hidden="true"></i>
</button>

Só ícone · contorno (menu / ColVis)

Botão quadrado com apenas borda e fundo transparente — útil para menu (fa-bars), ColVis (fa-ellipsis-v), pesquisa, idioma, anexos ou ações compactas na toolbar. Use btn btn-outline-icon em componentes.css e define sempre aria-label (ou texto visível em outro elemento associado).

Links com aspecto de botão

Use <a href="…"> com class="btn …" quando a ação for navegação (mudar de página, abrir URL externa, baixar arquivo). Para ações só em JavaScript na mesma tela, preferir <button type="button">. Opcional: role="button" no <a> quando o destino for tratado por script.

Todas as variantes «cheias» abaixo são <a> — o aspecto reproduz os botões equivalentes.

Primário (portal) Marca (band) Neutro (cancelar)
HTML 
<a href="@Url.Action("Detalhe", "Fatura", new { id = Model.Id })" class="btn btn-primary">Ver fatura</a>

<a href="https://exemplo.pt/ajuda" class="btn btn-band" target="_blank" rel="noopener noreferrer">Centro de ajuda</a>

<a href="@Url.Action("Index", "Home")" class="btn btn-canceled">Voltar à lista</a>

Links em estilo texto (btn-link)

Combina btn com btn-link para uma ação discreta, alinhada ao token --cor-action-texto (sem caixa preenchida). Funciona com <a> ou <button type="button">.

Saber mais Abrir documentação
HTML 
<a href="~/Ajuda/Termos" class="btn btn-link">Termos e condições</a>

<button type="button" class="btn btn-link">Mostrar menos</button>

<a href="@Url.Action("Exportar", "Relatorio")" class="btn btn-link">
    <i class="fa fa-download" aria-hidden="true"></i> Exportar relatório
</a>

Desativado

Em <button> use o atributo disabled. Em <a> combina class="btn … disabled" (ou aria-disabled="true") com tabindex="-1" e remove o href ou bloqueia o clique no controlador de eventos — um link com href continua acessível por teclado sem medidas extra.

Link inativo
HTML 
<button type="button" class="btn btn-primary" disabled>Indisponível</button>

<button type="button" class="btn btn-canceled" disabled>
    <i class="fa fa-ban" aria-hidden="true"></i> Bloqueado
</button>

<a class="btn btn-canceled disabled" href="#" tabindex="-1" aria-disabled="true">Inativo</a>

Menu pendente (dropdown)

Agrupa o botão e a lista num contêiner com data-invoisys-ds-dropdown. O gatilho leva data-invoisys-ds-dropdown-toggle, aria-expanded e aria-haspopup="true". Os itens podem ser <a> ou <button type="button"> dentro de <li>.

  • Editar
  • Duplicar
  • Remover
HTML 
<div class="invoisys-ds-dropdown" data-invoisys-ds-dropdown>
    <button type="button" class="btn btn-primary" data-invoisys-ds-dropdown-toggle
            aria-expanded="false" aria-haspopup="true" id="menu-acoes">
        Ações <i class="fa fa-caret-down" aria-hidden="true"></i>
    </button>
    <ul class="invoisys-ds-dropdown-menu" role="menu" aria-labelledby="menu-acoes">
        <li role="none"><a href="#" class="invoisys-ds-dropdown-item" role="menuitem">…</a></li>
    </ul>
</div>
