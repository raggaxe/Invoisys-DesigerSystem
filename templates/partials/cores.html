Visão geral

O sistema de cores da Invoisys segue a mesma ideia da documentação do Bootstrap 5.3 — Color: tokens semânticos em :root (main.css), reutilizáveis em CSS com var(--cor-…). Na tabela, a coluna Valor reflete o hex (ou rgba) definido no arquivo — útil para alinhar com Figma ou revisões de contraste.

Regra: em código novo, preferir sempre o nome semântico (--cor-brand, --cor-texto-heading, …) em vez de repetir hex nas views — assim, quando o token mudar no main.css, todo o portal atualiza.

Utilitários de fundo (tokens.css)

Alguns fundos expõem-se também como classes utilitárias (úteis em markup sem CSS sob medida):

.bg-success .bg-info .bg-danger

Estas classes mapeiam para variáveis do design system; ver definições em tokens.css.

Snippet CSS

CSS 
.titulo-pagina {
  color: var(--cor-texto-heading);
}

.barra-marca {
  background: var(--cor-brand);
  color: var(--cor-texto-sobre-brand);
}

.card-acao {
  background: var(--cor-action-fundo);
  color: var(--cor-action-texto);
  border: 1px solid var(--cor-action-border);
}
