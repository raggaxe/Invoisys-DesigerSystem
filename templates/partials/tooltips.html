No portal (ASP.NET MVC) os tooltips seguem em geral o Bootstrap 3 com jQuery: data-toggle="tooltip", data-placement (top, bottom, left, right) e title (ou data-original-title após init). Para o aspecto de cartão branco com texto e contorno verdes (como o botão Importar XML), use a classe invoisys-tooltip-brand em componentes.css no elemento externo do tooltip (ver snippets abaixo).

Bootstrap 5.3: a API muda para data-bs-toggle="tooltip", data-bs-placement, data-bs-title e data-bs-custom-class; o JavaScript expõe bootstrap.Tooltip (com Popper). Documentação oficial: Bootstrap 5.3 · Tooltips. Esta página de design system não carrega o bundle BS5 no index.html (evite conflitos com o markup legado); o exemplo interativo de posições e do «Importar XML» está num arquivo estático com CDN, embutido abaixo num iframe.

Rede: o iframe pede CSS/JS ao jsDelivr. Sem internet, o bloco não inicializa os tooltips — mantenha o snippet HTML/JS copiável para o seu ambiente.

Demo interativa (Bootstrap 5.3)

Passe o mouse sobre os botões: os quatro de posição e o «Importar XML» usam data-bs-custom-class="invoisys-tooltip-brand" (fundo branco, texto e borda verdes). No BS5 também existem start / end em vez de left / right quando quiser alinhamento à direção do texto.

Portal · Bootstrap 3 + jQuery

Marque o gatilho com data-toggle="tooltip" e data-placement. Para aplicar invoisys-tooltip-brand, passe um template cujo elemento raiz tenha class="tooltip invoisys-tooltip-brand" (o Bootstrap 3 injeta o conteúdo em .tooltip-inner). Use container: 'body' quando o tooltip estiver dentro de contêineres com overflow recortado.

HTML do gatilho 
<button type="button" class="btn btn-secondary"
        data-toggle="tooltip" data-placement="top"
        title="Importar NF-e a partir de um arquivo XML"
        data-testid="tooltip-importar-xml">
    Importar XML
</button>
JavaScript (init) 
var tpl =
    '<div class="tooltip invoisys-tooltip-brand" role="tooltip">' +
    '<div class="tooltip-arrow"></div>' +
    '<div class="tooltip-inner"></div>' +
    '</div>';

$('#meuBotao').tooltip({
    template: tpl,
    container: 'body',
    html: false
});

Bootstrap 5.3 · marcação e init

Depois de incluir o bootstrap.bundle.min.js (já com Popper), inicialize os nós com data-bs-toggle="tooltip" ou construa new bootstrap.Tooltip(elemento, { container: document.body }).

HTML + JS mínimo 
<button type="button" class="btn btn-light border"
        data-bs-toggle="tooltip"
        data-bs-placement="top"
        data-bs-custom-class="invoisys-tooltip-brand"
        data-bs-title="Importar NF-e a partir de um arquivo XML"
        data-testid="tooltip-importar-xml">
    Importar XML
</button>

<script>
document.querySelectorAll('[data-bs-toggle="tooltip"]').forEach(function (el) {
    new bootstrap.Tooltip(el, { container: document.body });
});
</script>

Posições

  • Bootstrap 3: data-placement="top" | bottom | left | right.
  • Bootstrap 5.3: data-bs-placement com os mesmos valores; start / end substituem left / right em modo LTR (ver documentação).
