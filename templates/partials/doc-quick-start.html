Quick start

Comandos para correr esta documentação localmente, a ordem dos <link> no MVC, um exemplo mínimo de view e ligações para descarregar os ficheiros do stack CSS (e utilitários do repositório doc).

Servidor local desta documentação

Na raiz do repositório Git (onde está app.py), com Python 3 e dependências instaladas:

Terminal 
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py

Por defeito o servidor escuta em http://127.0.0.1:8080 (altere com a variável de ambiente PORT). Abra / no browser para navegar pela documentação.

Arquivos CSS no _Layout.cshtml (MVC)

Copie estes quatro <link> para o layout (ou secção equivalente), na ordem indicada em Introdução — ordem de cascade nas views. Se a view não usar tabelas, pode omitir table.css.

Razor / HTML · só as folhas 
<link href="~/Content/Css/main.css" rel="stylesheet" />
<link href="~/Content/Css/tokens.css" rel="stylesheet" />
<link href="~/Content/Css/table.css" rel="stylesheet" />
<link href="~/Content/Css/componentes.css" rel="stylesheet" />

Pré-visualização mínima (só classes do DS)

Em MVC, o mesmo bloco vive na sua .cshtml depois de referenciar as folhas no layout.

Snippet completo (Lato + layout + view)

MVC — layout + exemplo de view 
<!-- _Layout.cshtml (ou seção de estilos): Lato + stack CSS -->
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Lato:ital,wght@0,100;0,300;0,400;0,700;0,900;1,400&display=swap" rel="stylesheet" />
<link href="~/Content/Css/main.css" rel="stylesheet" />
<link href="~/Content/Css/tokens.css" rel="stylesheet" />
<link href="~/Content/Css/table.css" rel="stylesheet" />
<link href="~/Content/Css/componentes.css" rel="stylesheet" />

<!-- Exemplo de corpo numa view (.cshtml) -->
<div class="main-content">
  <div class="page-content">
    <div class="header d-flex flex-wrap align-items-center justify-content-between gap-2 mb-3">
      <h2 class="heading-md mb-0">Título da página</h2>
      <button type="button" class="btn btn-primary">Ação principal</button>
    </div>
    <p class="text-label mb-3">Subtítulo ou descrição curta da área de trabalho.</p>
    <div class="d-flex flex-wrap gap-2">
      <button type="button" class="btn btn-secondary">Secundário</button>
      <button type="button" class="btn btn-canceled">Cancelar</button>
    </div>
  </div>
</div>
